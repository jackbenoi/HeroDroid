Google Play App Store [CMS]

To check our demo site please visit: http://googelplayappstore.info

About Our Product

  • Google Play Store CMS is an android market site specifically for Google Play Store. You can add apps / games / themes etc in this CMS based on your needs.
  • You can get details such as title, images, screenshots etc of a specific apps / games using our admin page.

Features

  • Allow to add your own app / games / themes and can automate to get details if the app was published in google play store.
  • Allow Developer / Users to submit apps.
  • Review Submitted apps / games.
  • Add or Upload apk file.
  • Add featured apps.
  • Responsive Design.
  • Google Analytics Ready.
  • Report Spam Contents.
  • Contact us page.
  • Unlimited Add Pages.
  • Advertisements Ready
  • And many more . . .

Technology Used

  • Built using Laravel 5.2 Framework
  • Coffeescript
  • AngularJS
  • Gulp
  • Bootstrap 4

Server Requirements

  • PHP >= 5.5.9
  • OpenSSL PHP Extension
  • PDO PHP Extension
  • Mbstring PHP Extension
  • Tokenizer PHP Extension
